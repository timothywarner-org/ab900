# Labs

This directory contains hands-on lab exercises for Segment 1.

## Available Labs

### Lab 1.1: Explore Microsoft 365 Admin Center
**Difficulty:** Beginner  
**Duration:** 15 minutes

Navigate the admin center and identify key administrative areas.

### Lab 1.2: Review Identity Configuration
**Difficulty:** Beginner  
**Duration:** 15 minutes

Examine user identities and authentication settings in Azure AD/Entra ID.

### Lab 1.3: Analyze License Assignments
**Difficulty:** Beginner  
**Duration:** 10 minutes

Understand license distribution and identify Copilot requirements.

## Lab Instructions

Each lab includes:
- Learning objectives
- Step-by-step instructions
- Expected outcomes
- Troubleshooting tips
- Additional challenges (optional)

## Prerequisites

- Access to Microsoft 365 tenant
- Administrative credentials
- Modern web browser
- Stable internet connection

## Lab Environment

These labs can be completed in:
- Production tenant (with caution)
- Trial/demo tenant (recommended)
- Microsoft 365 Developer tenant
- Hands-on lab environment
