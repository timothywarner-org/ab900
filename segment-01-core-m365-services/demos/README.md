# Demos

This directory contains demonstration materials for Segment 1.

## Available Demos

### Demo 1: Microsoft 365 Service Architecture
**Files:** `demo-01-m365-architecture.md`

Visual walkthrough showing how Microsoft 365 services interconnect and how data flows between services.

### Demo 2: User Lifecycle Management
**Files:** `demo-02-user-lifecycle.md`

Step-by-step demonstration of creating users, assigning licenses, configuring authentication, and managing access.

### Demo 3: Service Health Monitoring
**Files:** `demo-03-service-health.md`

Overview of the service health dashboard, reading advisories, and understanding service incidents.

## Running the Demos

1. Ensure you have access to a Microsoft 365 admin account
2. Follow the steps in each demo file
3. Adapt examples to your specific environment
4. Document any variations or issues encountered

## Prerequisites

- Microsoft 365 admin center access
- Azure AD/Entra ID admin center access
- Appropriate administrative permissions
- Demo tenant or non-production environment (recommended)
