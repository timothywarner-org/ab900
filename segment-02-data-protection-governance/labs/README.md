# Labs

This directory contains hands-on lab exercises for Segment 2.

## Available Labs

### Lab 2.1: Create and Test DLP Policies
**Difficulty:** Intermediate  
**Duration:** 20 minutes

Implement DLP policies to protect sensitive information and test with sample content.

### Lab 2.2: Configure Sensitivity Labels
**Difficulty:** Intermediate  
**Duration:** 20 minutes

Create and deploy sensitivity labels with encryption and protection settings.

### Lab 2.3: Implement Retention Policies
**Difficulty:** Intermediate  
**Duration:** 15 minutes

Establish retention policies for different content types and test retention behavior.

### Lab 2.4: Audit Copilot Activities
**Difficulty:** Intermediate  
**Duration:** 15 minutes

Enable and review audit logs for Copilot usage and compliance reporting.

## Lab Instructions

Detailed step-by-step instructions are provided in the main segment README.

## Prerequisites

- Microsoft Purview compliance portal access
- Compliance Administrator role or equivalent
- Sample documents for testing
- Understanding of data protection concepts
