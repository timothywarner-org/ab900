# Labs

This directory contains hands-on lab exercises for Segment 3.

## Available Labs

### Lab 3.1: Verify Copilot Prerequisites
**Difficulty:** Beginner  
**Duration:** 10 minutes

Ensure your environment meets all Copilot requirements.

### Lab 3.2: Assign Copilot Licenses
**Difficulty:** Beginner  
**Duration:** 10 minutes

Deploy Copilot licenses to pilot users and verify activation.

### Lab 3.3: Configure Copilot Policies
**Difficulty:** Intermediate  
**Duration:** 15 minutes

Set up administrative policies using Cloud Policy management.

### Lab 3.4: Test Copilot Experiences
**Difficulty:** Intermediate  
**Duration:** 20 minutes

Validate Copilot functionality across all Microsoft 365 apps.

### Lab 3.5: Monitor Copilot Usage
**Difficulty:** Beginner  
**Duration:** 10 minutes

Access and interpret Copilot analytics dashboards.

### Lab 3.6: Troubleshoot Copilot Issues
**Difficulty:** Intermediate  
**Duration:** 15 minutes

Diagnose and resolve common Copilot problems.

## Lab Instructions

Detailed step-by-step instructions are provided in the main segment README.

## Prerequisites

- Microsoft 365 admin access
- Copilot licenses available
- Test user accounts
- Microsoft 365 Apps installed
