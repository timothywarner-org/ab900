# Demos

This directory contains demonstration materials for Segment 4.

## Available Demos

### Demo 1: Agent Builder Walkthrough
Creating an agent from scratch, adding knowledge sources, configuring actions, testing and debugging, and publishing.

### Demo 2: Enterprise Agent Deployment
Governance workflow, approval process, distribution to users, and ongoing monitoring and management.

### Demo 3: Power Platform Integration
Agent + Power Automate scenarios, Agent + Power Apps integration, and end-to-end solution demonstrations.

### Demo 4: Security and Compliance
Implementing security controls, applying DLP policies, reviewing audit logs, and investigating incidents.

## Running the Demos

1. Access Copilot Studio
2. Have sample use cases prepared
3. Show both development and admin perspectives
4. Demonstrate governance controls
5. Include real-world scenarios

## Prerequisites

- Copilot Studio access
- Power Platform environment
- Sample knowledge base content
- Administrator permissions for governance demos
