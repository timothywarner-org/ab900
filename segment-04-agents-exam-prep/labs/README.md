# Labs

This directory contains hands-on lab exercises for Segment 4.

## Available Labs

### Lab 4.1: Create Your First Declarative Agent
**Difficulty:** Beginner  
**Duration:** 20 minutes

Build a simple FAQ agent using Copilot Studio.

### Lab 4.2: Build an Advanced Agent with Actions
**Difficulty:** Intermediate  
**Duration:** 25 minutes

Create an agent that performs actions using Power Automate plugins.

### Lab 4.3: Configure Agent Governance
**Difficulty:** Intermediate  
**Duration:** 15 minutes

Implement governance controls and security policies for agents.

### Lab 4.4: Integrate Agent with Power Automate
**Difficulty:** Intermediate  
**Duration:** 20 minutes

Create a workflow that leverages an agent's capabilities.

### Lab 4.5: Security Hardening for Agents
**Difficulty:** Advanced  
**Duration:** 20 minutes

Apply security best practices and test security controls.

### Lab 4.6: Agent Analytics and Optimization
**Difficulty:** Intermediate  
**Duration:** 15 minutes

Monitor performance and optimize agent responses based on analytics.

## Lab Instructions

Detailed step-by-step instructions are provided in the main segment README.

## Prerequisites

- Copilot Studio access
- Power Platform environment
- Basic understanding of conversational AI
- Agent creation permissions
