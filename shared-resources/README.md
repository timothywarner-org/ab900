# Shared Resources

This directory contains resources that are useful across all course segments.

## Directory Structure

### `/scripts`
PowerShell and other scripts for automation and administrative tasks.

**Examples:**
- License assignment scripts
- Bulk user provisioning
- Report generation
- Configuration management

### `/templates`
Templates for various administrative tasks and documentation.

**Examples:**
- DLP policy templates
- Sensitivity label schemas
- Retention policy templates
- Deployment checklists
- Communication templates

### `/references`
Reference materials, documentation, and quick guides.

**Examples:**
- PowerShell cmdlet references
- Microsoft Graph API examples
- Troubleshooting guides
- Best practices documentation
- Architecture diagrams

## How to Use These Resources

1. **Scripts**: Test scripts in a non-production environment first
2. **Templates**: Customize templates to match your organization's needs
3. **References**: Bookmark frequently used references for quick access

## Contributing

If you create useful resources during the course, consider contributing them back to this repository via pull request.

## Important Notes

- Always follow your organization's policies when using these resources
- Verify compatibility with your environment before implementation
- Keep security and compliance in mind
- Document any modifications you make
- Test thoroughly before production deployment
